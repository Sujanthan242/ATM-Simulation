# ATM Simulation

A simple command-line ATM banking system simulation written in C.

## Description

This program simulates basic ATM operations including checking balance, depositing money, and withdrawing money. It maintains a single account balance and provides a menu-driven interface.

## Features

- Check Account Balance
- Deposit Money
- Withdraw Money
- Exit the Program

## Requirements

- C Compiler (e.g., GCC)
- Standard I/O libraries (included with most C compilers)

## How to Compile

```bash
gcc main.c -o atm
```

## How to Run

```bash
./atm
```

## Usage

1. Run the program.
2. Choose an option from the menu (1-4).
3. Follow the prompts to perform operations.
4. Choose 4 to exit.

## Initial Balance

The account starts with a balance of $1000.00.
